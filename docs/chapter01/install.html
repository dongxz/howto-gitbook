安装

安装

1. 安装 NodeJS

GitBook 需要 NodeJS (v4.0.0以上版本) 的环境，Mac 用户可以使用 Homebrew 进行安装。

安装完成之后，可以使用下面的命令来检验是否安装成功。

$ node -v
v10.14.2

2. 安装 GitBook

使用 NPM 命令来安装 GitBook

$ npm install gitbook-cli -g

安装完成之后，可以使用下面的命令来检验是否安装成功。

$ gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3

3. 安装 Markdown 编辑器

Typora 是一款轻便、简洁的 Markdown 编辑器，支持即时渲染技术。

官方提供了 Mac、Windows、Linux 安装包，下载安装即可。
