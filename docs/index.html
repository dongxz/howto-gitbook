简介

Gitbook 是一个开源的、基于 Node.js 的命令行工具，它支持使用 Markdown 语法构建漂亮的图书，拥有丰富的、易用的插件，基于 GitBook 构建的静态网页可以使用 GitHub Page 进行拖管。
