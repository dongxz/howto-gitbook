一、文件说明

1.1 GitBook目录结构

.
├── book.json
├── README.md
├── SUMMARY.md
├── chapter01/
|   ├── README.md
|   └── section_01_01.md
|   └── section_01_02.md
└── chapter02/
    ├── README.md
    └── section_02_01.md
    └── section_02_02.md

1.2 README.md

一个约定俗成的概述文件。

1.3 SUMMARY.md

这个文件用于编排 GitBook 书籍的章节目录，它通过 Markdown 中的列表语法来表示书籍目录，执行 gitbook init 时会依据列表关系生成文件的父子关系。一个简单的示例：

# Summary

* [Introduction](README.md)
* 第一章
    * [1.1节](chapter01/1.md)
    * [1.2节](chapter01/2.md)
    * [1.3节](chapter01/3.md)
* 第二章
    * [2.1节](chapter02/1.md)
    * [2.2节](chapter02/2.md)
    * [2.3节](chapter02/3.md)
* 第三章
    * [3.1节](chapter03/1.md)
    * [3.2节](chapter03/2.md)
    * [3.3节](chapter03/3.md)

1.4 book.json

该文件主要用来管理书籍配置信息，GitBook未生成该文件，我们可以自行添加。

{
    "title": "本书的标题",
    "author": "书本的作者",
    "description": "书本的简单描述",
    "language": "zh-hans",
    "links": {
        "sidebar": {
            "Dongxz's Blog": "https://dongxz.github.io/"
        },
        "sharing": {
            "facebook": null,
            "google": null,
            "twitter": null,
            "weibo": null,
            "instapaper": null
        }
    },
    "plugins": [
        "livereload",
        "highlight",
        "search",
        "lunr",
        "sharing",
        "fontsettings",
        "theme-default"
    ]
}

二、参数说明

属性 描述 备注
title 本书的标题 简介 · GitBook 最佳实践
author 本书的作者
description 本书的简单描述 感觉无用
language 使用的语言 感觉无用
links 在左侧导航栏添加链接信息 左侧菜单上部会添加指定的链接
styles 自定义页面样式 未测试
plugins 本书启用的插件列表 用于开启、关闭插件
pluginsConfig 插件的属性配置 用于设置插件的个性化配置

本教程的 book.json

{
    "title": "GitBook 最佳实践",
    "description": "使用 GitBook + Typora + Git 来写作的最佳实践",
    "language": "zh-hans",
    "author": "dongxz",
    "plugins": [
        "-lunr",
        "-search",
        "-sharing",
        "-fontsettings",
        "code",
        "todo",
        "splitter",
        "search-pro",
        "advanced-emoji",
        "expandable-chapters",
        "back-to-top-button",
        "theme-fexa"
    ],
    "pluginsConfig": {
        "theme-fexa": {
          "logo": "/assets/logo.jpg",
          "favicon": "/assets/favicon.ico",
          "search-placeholder": "输入关键字搜索"
        },
        "code": {
          "copyButtons": true
        }
    },
  "variables": {
    "themeFexa": {
      "showHome": false,
      "showLogo": false,
      "showTitle": false,
      "showNavigator": false
    }
  }
}
