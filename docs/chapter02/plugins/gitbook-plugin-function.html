功能增强类插件

一、代码块

1.1 代码块添加行号和复制按钮

插件地址：https://www.npmjs.com/package/gitbook-plugin-code

GitHub地址：https://github.com/davidmogar/gitbook-plugin-code

{
    "plugins": ["code"],
    "pluginsConfig": {
        "code": {
            "copyButtons": false
        }
    }
}
  • 显示代码行号，单行代码无行号
  • 复制按钮可关闭

1.2 Todo 插件

插件地址：https://www.npmjs.com/search?q=gitbook-plugin-todo

GitHub地址：https://www.npmjs.com/package/gitbook-plugin-todo

{
    "plugins": ["todo"]
}

使用示例

* [ ]  write some articles
* [x]  drink a cup of tea

1.3 emoji表情

插件地址：https://www.npmjs.com/package/gitbook-plugin-advanced-emoji

GitHub地址：https://github.com/codeclou/gitbook-plugin-advanced-emoji

{
    "plugins": ["advanced-emoji"]
}

1.4 文字加底色

插件地址：https://www.npmjs.com/package/gitbook-plugin-emphasize

GitHub地址：https://github.com/GitbookIO/plugin-emphasize

{
    "plugins": ["emphasize"]
}

使用示例

This text is {% em %}highlighted !{% endem %}
This text is {% em %}highlighted with **markdown**!{% endem %}
This text is {% em type="green" %}highlighted in green!{% endem %}
This text is {% em type="red" %}highlighted in red!{% endem %}
This text is {% em color="#ff0000" %}highlighted with a custom color!{% endem %}

效果

二、其他

2.1 打赏插件

插件地址：https://www.npmjs.com/package/gitbook-plugin-donate

GitHub地址：https://github.com/willin/gitbook-plugin-donate

{
    "plugins": ["donate"],
    "pluginsConfig": {
        "donate": {
          "wechat": "例：/images/qr.png",
          "alipay": "http://blog.willin.wang/static/images/qr.png",
          "title": "默认空",
          "button": "默认值：Donate",
          "alipayText": "默认值：支付宝捐赠",
          "wechatText": "默认值：微信捐赠"
        }
    }
}
