目录导航类插件

目录导航类插件

一、左侧目录树导航

1.1 左侧目录树折叠

插件地址：https://www.npmjs.com/package/gitbook-plugin-expandable-chapters

GitHub地址：https://github.com/DomainDrivenArchitecture/gitbook-plugin-expandable-chapters

{
    "plugins": ["expandable-chapters"]
}

1.2 侧边栏宽度可拖拽

插件地址：https://www.npmjs.com/package/gitbook-plugin-splitter

GitHub地址：https://github.com/yoshidax/gitbook-plugin-splitter

{
    "plugins": ["splitter"]
}

二、页内标题导航

2.1 回到顶部

插件地址：https://www.npmjs.com/package/gitbook-plugin-back-to-top-button

GitHub地址：https://github.com/stuebersystems/gitbook-plugin-back-to-top-button

{
    "plugins": ["back-to-top-button"]
}

2.2 页内 TOC

插件地址：https://www.npmjs.com/package/gitbook-plugin-simple-page-toc

GitHub地址：https://github.com/stuebersystems/gitbook-plugin-simple-page-toc

{
    "plugins" : [ 
        "simple-page-toc" 
    ],
    "pluginsConfig": {
        "simple-page-toc": {
            "maxDepth": 3,
            "skipFirstH1": true
        }
    }
}

使用方法：在需要生成目录的位置添加 toc 标签 <!-- toc -->

注意：仅支持 h1, h2, h3，且不能跨级

参数说明

Name Type Default Description
maxDepth Number 3 Use headings whose depth is at most maxdepth.
skipFirstH1 Boolean true Exclude the first h1-level heading in a file.

2.3 页内悬浮 TOC

插件地址：https://www.npmjs.com/package/gitbook-plugin-page-toc-button

GitHub地址：https://github.com/stuebersystems/gitbook-plugin-page-toc-button

{
    "plugins" : [ 
        "page-toc-button" 
    ],
    "pluginsConfig": {
        "page-toc-button": {
            "maxTocDepth": 2,
            "minTocSize": 2
        }
    }
}

注意：仅支持 h1, h2, h3，且不能跨级

参数说明

Name Type Default Description
maxTocDepth Number 2 Maximal depth of headers (2 = h1 + h2 + h3). A value > 2 is not supported.
minTocSize Number 2 Minimal number of toc entries for showing the toc button.
