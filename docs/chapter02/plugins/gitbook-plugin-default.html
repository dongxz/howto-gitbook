默认插件

一、默认插件

  • livereload - 实时加载
  • highlight - 代码高亮
  • search - 全文检索功能，导航栏左上角
  • lunr
  • sharing - 分享功能，导航栏右上角
  • fontsettings - 字体设置，最上方的"A"符号
  • theme-default

二、视图增强

2.1 定制 Logo

在导航栏左上角插入Logo。

插件地址：https://www.npmjs.com/package/gitbook-plugin-insert-logo

GitHub地址：https://github.com/matusnovak/gitbook-plugin-insert-logo

{
    "plugins": ["insert-logo"],
    "pluginsConfig": {
        "insert-logo": {
            "url": "http://www.example.com/my-logo.png",
            "style": "background: none; max-height: 30px; min-height: 30px;"
        }
    }
}

2.2 Github图标

在右上角添加 github 图标。

插件地址：https://www.npmjs.com/package/gitbook-plugin-github

GitHub地址：https://github.com/GitbookIO/plugin-github

{
    "plugins": [ "github" ],
    "pluginsConfig": {
        "github": {
            "url": "https://github.com/your/repo"
        }
    }
}

2.3 高级搜索

支持中文搜索，在使用此插件之前，需要将默认的 search 和 lunr 插件去掉。

插件地址：https://www.npmjs.com/package/gitbook-plugin-search-pro

GitHub地址：https://github.com/gitbook-plugins/gitbook-plugin-search-pro

{
    "plugins": [
        "-lunr", 
        "-search", 
        "search-pro"
    ]
}

2.4 sharing-plus

插件地址：https://www.npmjs.com/package/gitbook-plugin-sharing

GitHub地址：https://github.com/GitbookIO/plugin-sharing

{
    "plugins": [
        "-sharing",
        "sharing-plus"
    ]
    "pluginsConfig": {
        "sharing": {
            "facebook": true,
            "twitter": true,
            "google": false,
            "weibo": false,
            "instapaper": false,
            "vk": false,
            "all": [
                "facebook", "google", "twitter", "weibo", "instapaper"
            ]
        }
    }
}
  • sharing配置中：true的代表直接显示在页面顶端，为false的不显示，不写默认为false
  • "all" 代表点击分享符号显示出来的所有分享按钮
